## 🛠️ What Is This?

A real e-commerce shop website for **Andy's Woodworks** — a craftsman who builds handcrafted solid pine furniture, outdoor pieces, and home décor by hand. Every product on this site is something Andy actually builds and sells. Prices are available on request.
